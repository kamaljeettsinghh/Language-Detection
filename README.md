# Language-Detection 
[Click Here to Open Live App ](https://language-detection-ml-npl.streamlit.app/) 

#  Project Overview
- This project detects the language of a given text using machine learning and NLP techniques.
- It is deployed as a Streamlit app for easy interaction.

#  Features
- Upload or input text to detect language.
- Supports multiple languages (list them if possible).
- Interactive web interface via Streamlit.
- Jupyter Notebook included for reproducibility.

#  Tech Stack
-Languages: Python
-Libraries: Pandas, NumPy, Scikit-learn, Streamlit
-Tools: Jupyter Notebook, Excel (data source)

#  Workflow
-Data collection and preprocessing (Excel dataset - cleaned text).
-Feature extraction (TF-IDF / Bag-of-Words).
-Model training and testing
-Deployment with Streamlit

#  Results
- Accuracy metrics (95%).
